# 🚀 Vloader (v1.0)

Vloader is a simple and fast Python-based video downloader that allows users to download videos from online platforms and save them for offline use.

> ⚠️ This is an early testing version (v1.0). More features and improvements are coming soon.

---

## ✨ Features

- 🎬 Download videos in multiple resolutions
- 🎧 Extract audio (MP3)
- ⚡ Fast and lightweight
- 🖥️ Simple and easy-to-use interface
- 🔗 Paste-and-download functionality

---
